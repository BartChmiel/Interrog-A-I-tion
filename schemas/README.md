# Schemas

Kontrakty danych dla backendu, promptow, raportow i testow.

Schematy sa wazne, bo wymuszaja przewidywalny format odpowiedzi modelu i ulatwiaja audyt.

Current schemas:

- `case.schema.json`
- `question.schema.json`
- `ai-suggestion.schema.json`
- `audit-event.schema.json`
- `grounding-pack.schema.json`
- `grounded-suggestion-decision.schema.json`
- `grounded-suggestions.schema.json`
- `indicator.schema.json`
