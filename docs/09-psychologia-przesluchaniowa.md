# Psychologia przesluchaniowa

## Kierunek

Projekt powinien czerpac z podejscia investigative interviewing, a nie z konfrontacyjnych technik nastawionych na przyznanie sie.

Najwazniejsze zalozenie psychologiczne: celem jest uzyskanie mozliwie dokladnych informacji przy poszanowaniu praw osoby, a nie maksymalizacja presji.

## Fundamenty metodyczne

1. Neutralnosc
   - pytania nie powinny sugerowac odpowiedzi,
   - system powinien oznaczac pytania ryzykowne,
   - AI powinno proponowac bezpieczniejsze formulacje.

2. Rapport
   - rozmowa powinna wspierac kontakt i zrozumienie,
   - system nie powinien promowac agresywnej konfrontacji,
   - styl pytan ma znaczenie dla jakosci odpowiedzi.

3. Wolna narracja
   - pierwsza relacja powinna dawac przestrzen na opis wlasnymi slowami,
   - pytania doprecyzowujace powinny przychodzic po narracji,
   - system powinien odrozniac narracje od doprecyzowania.

4. Pamiec i stres
   - niespojnosc nie oznacza automatycznie klamstwa,
   - roznice moga wynikac z pamieci, stresu, traumy, jezyka lub zmeczenia,
   - raport powinien mowic o materiale, nie o osobowosci.

5. Cross-validacja narracyjna
   - ten sam fakt mozna sprawdzac przez czas, miejsce, uczestnikow, zrodlo wiedzy i kolejnosc,
   - pytania kontrolne musza byc neutralne,
   - celem jest doprecyzowanie, nie pulapka.

## Kategorie obserwacji

System moze oznaczac:

- brak odpowiedzi,
- odpowiedz niejasna,
- zmiane szczegolu,
- konflikt chronologiczny,
- brak zrodla wiedzy,
- temat nieporuszony,
- roznice wzgledem materialu sprawy,
- pytanie potencjalnie sugerujace.

System nie powinien oznaczac:

- klamstwa,
- manipulacji jako faktu,
- winy,
- zaburzen psychicznych,
- profilu osobowosciowego.

## Bezpieczne wskazniki

Dopuszczalne:

- pokrycie tematow,
- spojnosc chronologii,
- stabilnosc szczegolow,
- liczba luk,
- liczba elementow do weryfikacji,
- neutralnosc pytan,
- jakosc uzasadnienia sugestii AI.

Niedopuszczalne:

- procent klamstwa,
- indeks winy,
- ocena prawdomownosci osoby,
- automatyczny scoring podejrzanego,
- psychologiczna diagnoza osoby.

## Wzorce pytan

Preferowane:

- "Prosze opisac to zdarzenie wlasnymi slowami."
- "Co wydarzylo sie potem?"
- "Skad Pan/Pani to wie?"
- "Kto jeszcze mogl to widziec?"
- "Gdzie dokladnie to mialo miejsce?"
- "Czy moze Pan/Pani doprecyzowac godzine?"

Ryzykowne:

- "Czyli widzial Pan/Pani, ze on to zrobil?"
- "Dlaczego Pan/Pani ukrywa ten fakt?"
- "Przeciez to bylo o 20:00, prawda?"
- "Czy nie jest tak, ze zmienil Pan/Pani wersje?"

## Zasada implementacyjna

Kazdy modul psychologiczny powinien dzialac jako wsparcie procesu, a nie jako narzedzie diagnozy osoby.
