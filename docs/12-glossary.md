# Glossary

## Advisory Indicator

A non-binding signal that helps the operator inspect material. It is not a verdict.

## Audit Chain

Append-only audit event sequence where each event stores a hash of the previous event.

## Evidence Alignment

An advisory view showing how accepted material-question links support or fail to support case topics.

## Grounding Context Pack

A bounded, source-linked context package sent to the model-client boundary for grounded suggestion generation.

## Model Artifact

A workspace-local prompt, context, output, cache item, or evaluation item written under `models/`.

## Model Artifact Manifest

Hash-chained JSON manifest that records workspace-local model artifacts.

## Provenance

Traceable metadata showing how an output was produced, including prompt version, hashes, source ids, model id, and human decisions.

## Workspace

Per-case local filesystem boundary containing imports, sessions, exports, audit data, and model artifacts.

## Truthfulness Verdict

An automated conclusion that a person is lying or telling the truth. This is explicitly out of scope.
