# Frontend

Przyszly interfejs aplikacji.

Zakladane widoki:

- lista spraw,
- pulpit sprawy,
- plan pytan,
- tryb przesluchania,
- analiza po przesluchaniu,
- raport i audyt,
- ustawienia lokalnego modelu oraz trybu bezpiecznego.

UI powinien byc spokojny, operacyjny i zbudowany pod dluga prace, nie pod efektowna prezentacje marketingowa.
